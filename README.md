# Antarctica.io

A modern, glassmorphic Vue.js application built with Tailwind CSS and Vite.

## Features

- **Vue.js 3.5** - Progressive JavaScript framework
- **Tailwind CSS v4** - Utility-first CSS framework
- **Vite 7** - Next generation frontend tooling
- **Glassmorphic Design** - Modern UI with glass effects and animations
- **Material Symbols** - Google's Material Design icons
- **Custom Arctic Color Palette** - Beautiful blue gradient theme

## Project Structure

```
antarctica.io/
├── assets/              # Static assets and prototypes
├── public/              # Public static files
├── src/
│   ├── App.vue         # Main application component
│   ├── main.js         # Application entry point
│   └── style.css       # Global styles and custom CSS
├── index.html          # HTML entry point
├── vite.config.js      # Vite configuration
├── tailwind.config.js  # Tailwind CSS configuration
└── postcss.config.js   # PostCSS configuration
```

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/chevp/antarctica.io.git
cd antarctica.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Design System

### Colors

The project uses a custom Arctic color palette:
- Arctic 50-900: Custom blue gradient colors
- Slate 50-950: Base UI colors
- Additional accent colors (purple, green) for cards

### Typography

- **Font Family**: Inter (Google Fonts)
- **Weights**: 300, 400, 500, 600, 700, 800

### Components

- **Glass Cards**: Frosted glass effect with backdrop blur
- **Hover Animations**: Smooth transitions and glow effects
- **Grid Background**: Subtle grid pattern overlay
- **Diagonal Stripes**: Pattern for header and footer
